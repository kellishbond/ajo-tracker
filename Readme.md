# Ajo Tracker 🚀

A digital version of the traditional African rotating savings system (Ajo/Esusu/Chama), built to manage fixed contribution cycles, secure rotations, and financial tracking.

## 🛠️ Tech Stack
- **Backend:** Go (Golang) + Gin Framework
- **Frontend:** React + Tailwind CSS
- **Database:** PostgreSQL

## 📈 Status
Currently **Building in Public**. Day 1 foundation (Server + Database connection) is complete.
